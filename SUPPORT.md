# Support

This project is maintained under the T50-Systems organization defaults. Use the support path that best matches your need.

## Bugs

Open a bug report in this repository and include reproduction steps, expected behavior, actual behavior, logs, environment details, and the affected version or commit.

## Feature requests

Open a feature request in this repository. Explain the user problem, the desired outcome, alternatives considered, and any compatibility or migration concerns.

## Questions

If this repository has Discussions enabled, use Discussions for questions. Otherwise open an issue labeled as a question if the project supports that workflow.

## Security

Do not use public issues for security reports. Follow [SECURITY.md](SECURITY.md).

## Scope

Maintainers may close requests that are out of scope, inactive, missing required information, or better handled in another repository.
